# Personal Portfolio Website

## Overview

This project is a responsive Personal Portfolio Website developed using HTML and CSS. The website serves as a professional online portfolio to showcase personal information, technical skills, projects, resume, and contact details. It is designed with a clean and modern user interface that works smoothly across desktop and mobile devices.

## Features

* Responsive navigation bar with smooth scrolling.
* Attractive hero section with introduction and call-to-action button.
* About Me section highlighting professional background.
* Skills section displaying technical competencies.
* Projects section showcasing completed projects.
* Resume download functionality.
* Contact section with email, phone, LinkedIn, and GitHub details.
* Mobile-friendly design using CSS media queries.
* Interactive hover effects and modern card layout.

## Technologies Used

* HTML5
* CSS3
* Responsive Web Design
* Flexbox
* CSS Grid
* Media Queries

## Project Structure

* Navigation Menu
* Hero Section
* About Section
* Skills Section
* Projects Section
* Resume Section
* Contact Section
* Footer

## Learning Outcomes

Through this project, I gained practical experience in:

* Creating structured web pages using HTML.
* Designing responsive layouts with CSS.
* Implementing Flexbox and Grid for page organization.
* Applying hover effects and transitions for better user experience.
* Building a professional portfolio website suitable for showcasing skills and projects.

## Conclusion

This portfolio website demonstrates fundamental front-end web development skills and responsive design principles. It provides a professional platform for presenting personal achievements, technical skills, and project work while ensuring accessibility across various devices.
